# LLM Slidev 演示文稿

这是一个使用 Slidev 制作的关于大语言模型（LLM）的演示文稿项目。

## 内容概览

本演示文稿涵盖以下主题：

1. **LLM 简介** - 什么是大语言模型
2. **核心技术** - Transformer、预训练方法
3. **能力展示** - 语言理解、生成、推理
4. **应用场景** - 实际应用案例
5. **挑战与未来** - 当前挑战和发展方向

## 安装

```bash
npm install
```

## 开发

启动开发服务器：

```bash
npm run dev
```

然后在浏览器中打开 http://localhost:3030

## 构建

生成静态网站：

```bash
npm run build
```

生成的文件将在 `dist/` 目录下。

## 导出

导出为PDF：

```bash
npm run export
```

## 自定义

编辑 `slides.md` 文件来修改演示内容。

更多信息请访问 [Slidev 官方文档](https://sli.dev/)
